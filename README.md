# CLINIANI AI

A modern Neurology EMR built using Node.js, Express, MongoDB and OpenAPI.

## Features

- Patient Management
- JWT Authentication
- Refresh Tokens
- Role Based Access Control
- Swagger Documentation
- MongoDB
- REST APIs

## Technology Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt
- Swagger
- Postman

## Completed Modules

- Authentication
- Authorization
- Patient CRUD
